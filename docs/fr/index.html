Laisse Moi Faire

Ce logiciel permet de faciliter la maintenance à distance sur des postes sous Linux.

Une fois lancé avec les droits administrateurs :

  • Ouvre le port SSH
  • Donne un accès VNC
  • Crée l'administrateur let-me-do

Installation

Ubuntu zesty, yakkety ou xenial

sudo add-apt-repository ppa:1o1o22/let-me-do
sudo apt-get update
sudo apt-get install let-me-do

Archlinux

yaourt -S let-me-do
